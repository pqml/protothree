# Protothree
:dart: Three.js boilerplate for rapid and modulable prototyping

<br>

### Features
  - Webpack 2
  - `glslify-loader`
  - Hot reloading on shaders
  - Loader utils to preload textures or objects
  - `ghp` to quickly deploy a demo to a gh-page
  - Raf singleton with deltatime
